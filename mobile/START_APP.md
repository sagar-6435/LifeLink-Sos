# How to Start the App

## Quick Start (No Maps)

```powershell
cd mobile
npx expo start --clear
```

Press `a` for Android or scan QR code.

Maps will show a placeholder - app works fine!

---

## Full Start (With Maps)

```powershell
cd mobile
npx expo run:android
```

First build takes 5-10 minutes. Maps will work!

---

## That's It!

Choose based on what you need to test:
- **Testing UI/features?** → Use `expo start`
- **Testing maps?** → Use `expo run:android`
